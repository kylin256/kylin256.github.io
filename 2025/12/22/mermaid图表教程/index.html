Mermaid 图表教程

  • Mermaid 是一个基于 JavaScript 的图表绘制工具，它使用简单的文本描述语法来生成各种图表。你可以直接在 Markdown 文件中编写 Mermaid 代码，许多平台（如 GitHub、GitLab、Obsidian、VS Code 等）都支持直接渲染。

基本语法结构

在 Markdown 中使用 Mermaid，需要用mermaid 代码块包裹：

注意，不需要反斜杠，这里为了防止被转化才写
\`\`\` mermaid
graph TD
    A[开始] --> B(处理)
    B --> C{判断}
\`\`\`

类关系符号

<|-- : 继承
*-- : 组合
o-- : 聚合
--> : 关联
-- : 实线连接
..> : 依赖

小贴士

  • 注释: 使用 %% 添加注释
  • 样式: 使用 style关键字自定义样式
  • 链接: 可以为节点添加点击链接
  • 多行文本: 使用 <br> 换行
  • 主题: 支持多种主题（默认、森林、黑暗等）
